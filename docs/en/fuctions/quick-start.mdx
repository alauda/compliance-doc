# Quick Start

## Prerequisites

- Alauda Container Platform version: >= v4.1.0
- Plugin version: >= v1.1.0

## Quick Scan

After the plugin is installed, you can quickly create scans using the following templates.

### K8S STIG Scan

```yaml
apiVersion: compliance.security.alauda.io/v1alpha1
kind: ScanSuite
metadata:
  name: stig-scan
  namespace: compliance-system
spec:
  profiles:
    - stig-k8s-api-server
    - stig-k8s-control-plane
    - stig-k8s-general
    - stig-k8s-kubelet
```

### K8S CIS Scan

```yaml
apiVersion: compliance.security.alauda.io/v1alpha1
kind: ScanSuite
metadata:
  name: cis-scan
  namespace: compliance-system
spec:
  profiles:
    - cis-k8s-controlplane
    - cis-k8s-etcd
    - cis-k8s-master
    - cis-k8s-node
    - cis-k8s-policies
```

### MicroOS STIG Scan

```yaml
apiVersion: compliance.security.alauda.io/v1alpha1
kind: Scan
metadata:
  name: os-scan
  namespace: compliance-system
spec:
  profile: stig-os-microos
```
