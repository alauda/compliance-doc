# Upgrade

:::info

This document describes patch-level upgrade guidance for Alauda Compliance Service.

:::

## Product Compatibility

The table below lists product compatibility for Compliance Service patch-level upgrades:

| Compliance Service Version | Supported ACP Version |
| --- | --- |
| 1.1.1 | 4.1.x, 4.2.x, 4.3.x |
| 1.1.0 | 4.1.x, 4.2.x |

:::note
Email notification is not supported on ACP 4.1.x.
:::

## Patch-Level Upgrade

- **Description**: Upgrades between patch versions within the same minor version are fully compatible and can be performed directly for Compliance Service.
- **Scope**: This guidance applies to patch-level updates only and does not cover minor or major version upgrades.
- **Example**: `1.1.0 -> 1.1.1`
